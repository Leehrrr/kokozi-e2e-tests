# kokozi-e2e-tests

Playwright E2E test for kokozi.com cart merge scenario

---

## 사용 도구

| 도구 | 용도 |
|------|------|
| **Python 3.13** | 테스트 작성 언어 |
| **Playwright** | 브라우저 자동화 (Chromium) |
| **pytest** | 테스트 실행 프레임워크 |
| **pytest-playwright** | pytest ↔ Playwright 연동 |

---

## 테스트 시나리오

**CRT-MG: 비회원 장바구니 로그인 시 병합**
> 비회원 상태에서 담은 상품이 로그인 후 회원 장바구니와 합쳐지는지 검증

### 사전 조건

1. 회원 계정으로 로그인
2. 기존 장바구니 전체 비우기
3. 상품 B (핑크퐁 베베핀 아띠) 장바구니 담기
4. 로그아웃

### 테스트 스텝

| ID | 스텝 | 기대 결과 |
|----|------|-----------|
| CRT-MG-01 | 비회원 상태에서 상품 A (아기상어 아띠) 장바구니 담기 | 장바구니에 상품 A 담김 |
| CRT-FL-05 | 헤더 장바구니 메뉴 열기 | 상품 A 노출 확인 |
| CRT-MG-02 | 회원 계정으로 로그인 후 장바구니 이동 | 상품 A + 상품 B 모두 노출 확인 |
